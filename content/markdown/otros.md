### Texto Enriquecido

Formatea texto fácilmente. Ejemplos:

- **Negrita**: ``` **texto** ``` → **texto**
- *Cursiva*: ``` *texto* ``` → *texto*
- ~~Tachado~~: ``` ~~texto~~ ``` → ~~texto~~

### Listas

Enumera una serie de elementos y hay varios tipos:

- **Listas con viñetas**:

```
- Elemento 1
- Elemento 2
    - Sub-elemento 2.1
```

- Elemento 1
- Elemento 2
    - Sub-elemento 2.1


- **Listas con números**:

```
1. Primer elemento
2. Segundo elemento

```

1. Primer elemento
2. Segundo elemento
