3. Primary Menu

The sidebar of the theme uses the main menu. Add all the items for your sidebar navigation in this menu.

// config.toml

[menu]
[[menu.main]]
  name = 'Home'
  url = '/'
  weight = 1
[[menu.main]]
  name = 'Content'
  params.header = true
  weight = 2
[[menu.main]]
  name = 'Articles'
  url = '/articles'
  weight = 3
